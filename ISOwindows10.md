# como baixar ISO windows 10
- o que é ISO ?<br>
um arquivo que contem uma **cópia exata** de um sistema de arquivo de um disco óptico (cd/dvd)<br>
permite instalar o ISO (sistema operacional) em uma maquina fisica ou virtual.<br>
